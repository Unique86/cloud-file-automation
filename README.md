Cloud File Automation Tool

A web-based file conversion platform built with FastAPI.

• Convert images to PDF
• Merge PDF files
• File storage and download system
• Mobile responsive interface

Tech Stack
Python
FastAPI
JavaScript
HTML/CSS

Goal
To demonstrate backend API design, file processing automation,
and responsive web application development.
